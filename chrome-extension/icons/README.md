# Chrome Extension Icons

Generate icons using any tool or the included \`generate_icons.html\` file:
- icon16.png: 16x16px
- icon32.png: 32x32px
- icon48.png: 48x48px
- icon128.png: 128x128px

Recommended design:
White shield outline on dark navy (#0a0f1e) background with cyan (#00ffff) accent.

Quick option: Use [favicon.io](https://favicon.io) to generate from text 'A' with dark background.
